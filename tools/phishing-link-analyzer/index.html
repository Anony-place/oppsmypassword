How This Tool Works

This analyzer performs client-side heuristic checks on incoming URLs. It looks for classic indicators of phishing infrastructure, including:

  • IP Addresses: Legit services use domains, not raw IPs.
  • Subdomain Stuffing: Extremely long sets of subdomains (e.g., secure.login.verify.update.paypal.com.scammer.tk).
  • High-Risk TLDs: Top-Level Domains that are disproportionally abused due to free registration (e.g., .tk, .ml).
  • Homoglyphs and Hyphens: Using dash-stuffing to look official.

Note: A "Low Risk" score does not mean the site is 100% safe. Sophisticated attackers can host phishing pages on entirely legitimate, compromised domains. Always use Multi-Factor Authentication.
